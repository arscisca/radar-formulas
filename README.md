# radar-formulas
A collection of useful formulas for the Radar and Remote Sensing course. See **main.pdf** for the formulas.

## Disclaimer
Please notice that all the documents in this repository are completely unofficial and unrelated to any professor, assistant or course representative. For this reason there could be errors in the shared formulas and it is in the reader's own interest to check their validity.
These documents are online just for the convenience of sharing it with other students and to simplify the process of maintenance.
